
## Using the Enhanced System

### Basic Features

- **Start/Stop Camera**: Use the buttons in the interface to start or stop the camera.
- **Video Recording**: Click the "Record" button to start recording video, and click again to stop.
- **Image Analysis**: You can upload images to analyze them using the "Analyze Image" button.
- **Capture Image**: Click "Capture Image" to take a snapshot from the camera and save it.

### Enhanced Features

- **Dual Emotion Analysis**: The system can now detect and display dual emotions (e.g., Happiness with Surprise).
- **Higher Accuracy**: Emotion recognition accuracy is improved using multiple models.
- **Advanced Image Processing**: Image processing is enhanced to achieve better results in various lighting conditions.
- **Faster Response**: The system is now more responsive to changes in facial expressions.

## Understanding Analysis Results

When running the system, you will see the following on the screen:

1. **Green Rectangle**: Indicates the detected face region.
2. **Main Text**: Displays the primary detected emotion and its confidence (e.g., "Happy (0.85)").
3. **Additional Texts**: Show any detected dual emotions and their confidence.
4. **Different Colors**: Each emotion has a unique color for easier recognition.

## Troubleshooting

### Issue: Camera Not Working

- Ensure the camera is properly connected.
- Make sure no other application is using the camera.
- Try changing the camera number in the code (from 0 to 1 or 2).

### Issue: Low Emotion Recognition Accuracy

- Improve lighting in your environment.
- Make sure your face is clearly visible in the camera.
- Avoid extreme angles or being too far from the camera.

### Issue: Installation Errors

- Ensure all required libraries are installed.
- Try installing libraries individually.
- Check library version compatibility.
